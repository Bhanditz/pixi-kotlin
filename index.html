These are Pixi.js examples from http://www.pixijs.com/, ported to Kotlin. Each example contains links to the original example, the original source, the Kotlin source and the JS compiled from Kotlin actually running in the example.

On GitHub: abesto/pixi-kotlin

Supported versions:

  • Pixi.js: 2.2.5
  • Kotlin: M12.1 (0.12.613)
